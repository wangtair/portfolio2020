Portfolio 8 - 3D Related Animations and Models

[MI 247 Final] - Kyogre Model

Start Date: Late November, 2019

End Date: Dec. 12th, 2019

Place of Production: the College of ComArtSci, Michigan State University

Team Member: None

Engine: Maya, Z-Brush, Substance Painter, Substance B2M, and Unity 3D

Project type: Class Project

Game Type: 3D Model

What I was responsible for: The Whole Model and Effect

This 3D Kyogre is the final project of the MI 247 course last semester. I firstly created the mesh model by Z-Brush. Then I edited the poly model in Maya and used Substance Painter to draw its texture. Finally, I put the model into the Unity 3D to form light effects. The image is screenshot by Unity 3D.

[MI 247 Project 2] - House Model

Start Date: Oct. 28th, 2019

End Date: Nov. 5th, 2019

Place of Production: the College of ComArtSci, Michigan State University

Team Member: None

Engine: Maya, Substance B2M, and Unity 3D

Project type: Class Project

Game Type: 3D House Model

What I was responsible for: The Whole Model and Effect

This house is the second project of the MI 247. I created the model of the house by Maya. Then I did UV unwrapping inside Maya. Finally, I adjusted the material and light effects in Unity 3D. This work was on the exhibition of the Project Review.

3D Animation Music Video - METEOR

Start Date: Mid April, 2018

End Date: Early May, 2018

Place of Production: Fahy Hall, Seton Hall University

Team Member: None

Engine: Maya, AfterEffect, Premiere

Project type: Class Project

Game Type: 3D Animation Music Video

What I was responsible for: The Whole Animation

This music video is the final project of the 3D animation class I’ve taken in Seton Hall University in 2018. The full music video was created by drafting, referencing, animation creation, and final video editing. The BGM is “METEOR” created by Hitoshi Sakimoto in Gradius V. The inspiration of this music video is some music videos created by Bemani.
