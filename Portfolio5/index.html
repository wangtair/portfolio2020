Portfolio 5 - [CAS 117 Project 4] Box Shooter

This is the fourth game project of the CAS 117 course. It’s also the final project of this class.

The game structure is referred to the Dr. Winn’s tutorial and most of the scripts are supported by him. So, this game is mainly about the utilization of the Unity 3D. Except from the settings of the levels and system, I’ve added some new systems such as moving objects and hazards. I borrowed some scripts from the third project to add new mechanics.
